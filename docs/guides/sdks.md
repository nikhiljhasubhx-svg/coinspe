# SDKs & Tools

Official SDKs streamline authentication, retries, serialization, and idempotency.

| Language | Package | Status |
| --- | --- | --- |
| JavaScript/TypeScript | `@coinspe/sdk` | GA |
| Python | `coinspe-sdk` | Beta |
| Go | `github.com/coinspe/coinspe-go` | Alpha |

## Installing

=== "JavaScript"
```bash
npm install @coinspe/sdk
```

=== "Python"
```bash
pip install coinspe-sdk
```

=== "Go"
```bash
go get github.com/coinspe/coinspe-go
```

## CLI helper

```
npx coinspe-cli login
npx coinspe-cli payments create --amount 100 --currency USD
```
