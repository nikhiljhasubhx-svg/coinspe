# Support

Need help integrating Coinspe? Choose the path that matches your issue severity.

| Channel | Use cases | SLA |
| --- | --- | --- |
| Email (support@coinspe.com) | Non-urgent questions, sandbox setup | 1 business day |
| Dashboard chat | Production-impacting but not blocking | 4 hours |
| Pager hotline | Critical live incidents | 15 minutes |

When opening a ticket, include:

- `request_id` and timestamp
- Environment (sandbox or production)
- Endpoint + payload snippet (remove PII)
- Logs or screenshots

Security incidents should go straight to `security@coinspe.com` with encrypted attachments.
